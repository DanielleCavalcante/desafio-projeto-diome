# Técnicas de Programação

#### Linear:

- Modelo tradicional
- Não tem vínculo
  - Estrutura hierárquica 
  - Programação de computadores

- Execução estruturada
- Recursos Limitados
- Única dimensão

#### Estruturada:

- Não é linear 
- Permite utilizar mais de uma opção (escolhas)

#### Modular:

- Se dá pela definição de partes independentes que são controladas por um conjunto de regras.
- **Benefícios:**
  - Simplificação
  - Decomposição do problema
  - Verificação do problema pode ser por etapas
